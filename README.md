# T-banner2

# Instalación 

$ pkg install python2 -y

$ pkg install cowsay -y

$ pkg install git -y

$ git clone https://github.com/Anonymous-Zpt/T-banner2

$ cd T-banner2 

$ chmod +x *

$ ./start
